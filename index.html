Easy Bible passage preview on most any webpage

It can be a hassle having to look up verses one by one as you see them in an online article.

I made a bookmarklet to make this easier.

Usage

  1. Select a translation:
  2. Drag this link to your bookmarks bar: RefTagger
  3. Click the bookmark when you're on a webpage with un-linked Bible references. Each reference will then become a link that you can mouse-over to preview the scripture passage.
